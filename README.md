## Opinions

* Take care of browser prefixing in post-processing.
* Generally follow the airbnb CSS [style guide](https://github.com/airbnb/css)

## Install

```
$ bower install shades
```

or

```
$ bower install git://github.com/emilyhorsman/shades.git
```

## Lint

```
$ bundle install
$ scss-lint
```
